Text File Handling 16/05/2017
Text File Handling написана на языке С

Описание программы
---------------------------------------------------------------------
Данное приложение создает текстовый файл >10000 символов; обрабатывает 
его; линкует и запускает пользовательское решение; сравнивает 
результаты и выдает ответ.

Установка под Linux системами
---------------------------------------------------------------------
Папка с программой Text File Handling должна содержать 4 папки:
	1) папка generator с программой генерирования текста.
	2) папка refsol с программой обработки текста
	3) папка result с файлами решения расширения .txt
	4) папка test с сгенерированными файлами расширения .txt
а также Ваше решение и bash-файл run.sh

Минимальные требования для программы Text File Handling
---------------------------------------------------------------------
1) Ваше решение должно быть написано на языке С.
2) Ваше решение должно иметь имя usersol.c.
3) На вход (из входного потока) Ваша программа должна получать сначала
имя_файла.txt, потом строку команд.
	Команды:
	1 - считает количество слов в файле
	2 - удаляет все слова, в которых больше гласных, чем согласных
	3 - сортирует слова в порядке уменьшения количества букв
	4 - считает количество предложений в файле
	5 - сохраняет результат в новом файле
4) На выход Ваша программа должна выдавать файл userout.txt, содержащий
значения (если они есть) в таком порядке:
	1 - количество слов
	2 - количество предложений
	3 - измененный текст
В случае, если на вход подается лишь команда 5, создается пустой файл.
В случае, если дважды встретилась команда 2, в выходном файле пишется 
"ERROR".

P.S. Не забудьте, что текст создается с отступами и знаками препинания: 
".", "!", "?", "?!", "...", "(", ")", "-", ":", ";", ",". Дефис не 
считается разделением слова, но он не учитывается при подсчете букв.

Запуск программы Text File Handling
---------------------------------------------------------------------
В терминале вызовите функцию:
	
	bash ./ran.sh
